cntRead

cntRead — Read current value of a counter object.

Description

Plugin opcode in counter.

Read the current value in a counter object without changing it.

Syntax

kval cntReadicnt

Initialization

icnt -- a handle for the counter.

`

Performance

Reads a counter object.
