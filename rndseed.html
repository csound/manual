rndseed

rndseed — Sets the global rndseed value.

Description

Sets the global seed value for rnd and birnd. 

rndseed ival

Performance

Use of rndseed will provide alternative results from the rnd and birnd opcodes to the default which is a seed of 0.5.

When specifying a seed value, ival should be between 0 and 1.
