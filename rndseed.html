rndseed

rndseed — Sets the global rndseed value.

Description

Sets the global seed value for rnd and birnd.

Syntax

rndseed ival

Initialization

ival (default=0.5) -- should be between 0 and 1.

Use of rndseed will provide alternative results from the rnd and birnd opcodes to the default (= 0.5).

Examples

Here is an example of the rndseed opcode. It uses the file rndseed.csd.

Example 905. Example of the rndseed opcode.

<CsInstruments>

; by Menno Knevel 2021

ksmps = 32

printk2 kbin
asig    vco2    .5, 100
ares    rezzy   asig, 1500+kbin, 20
        outs ares, ares
endin

</CsInstruments>
