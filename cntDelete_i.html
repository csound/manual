cntDelete_i

cntDelete_i — Delete a counter.

Description

Plugin opcode in counter.

Delete a counter and render any memory used.

Syntax

ival cntDelete_i icnt

Initialization

icnt -- the handle of a counter object from a call to cntCreate.

Performance

ival -- the handle deleted or a negative number if there was no such counter.

See Also

cntCreate, cntDelete, count, count_i, cntCycles, cntRead, cntReset, cntState
