Software Bus

Csound implements a software bus for internal routing or routing to external software calling the Csound API.

The opcodes to use the software bus are:

  • chn_k

  • chn_a

  • chn_S

  • chnclear

  • chnexport

  • chnget

  • chnmix

  • chnparams

  • chnset
