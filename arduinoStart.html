arduinoStart

arduinoStart — Open a serial port for use with the Arduino protocol.

Description

Plugin opcode in serial.

Open a serial port for arduino and use the Csound-Arduino protocol.

Syntax

iPort arduinoStart SPortName [, ibaudRate]

Initialization

SPortName -- port name number

ibaudrate -- serial speed, defaulting to 9600 bps.
