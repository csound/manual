Building Csound

Csound has become a complex project and can involve many dependencies. Unless you are a Csound developer or need to develop Csound plugins, you should try to use one of the precompiled distributions from https://csound.com/download.

Detailed and up to date information about building Csound from source can be found in the BUILD.md file in the Csound6 sources.
